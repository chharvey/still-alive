The Box Object

Use the Box object for stand-alone, highlighted blocks. They are useful for things like alerts, pop-ups, and asides. Since they have a border radius, they don’t look good in tables or table-like structures.

The Textbox Object

The Textbox Object is similar to the Box object except that it is used for inline text only, such as labels or badges. It is usually instantiated by a small element, but that depends on context.

The Golden Container

See my article griddemo.html for docs on the GoldenContainer object.

The following is a .o-GoldenContainer Object, containing 3 children:

  1. a .o-GoldenContainer__Content Object
  2. a .o-GoldenContainer__Sidebar.o-GoldenContainer__Sidebar--major Object
  3. a .o-GoldenContainer__Sidebar.o-GoldenContainer__Sidebar--minor Object

Sometimes you may have objects inside the Content that you wish to make appear as a sidebar or as if it weren’t in the Content.

The following .o-GoldenContainer Object contains only a .o-GoldenContainer__Content Object, which contains 3 children:

  1. a .o-GoldenContainer__Content__Page Object
  2. a .o-GoldenContainer__Content__SideMajor Object
  3. a .o-GoldenContainer__Content__SideMinor Object

Even though these are children of .o-GoldenContainer__Content, they appear as if they are outside (for example, inside a __Sidebar--major or __Sidebar--minor).
