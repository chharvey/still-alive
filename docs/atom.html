Font Families

Font Sizes

The following atoms adjust font size. All atoms use em units, so the font sizes will scale according to their parents.

The first list shows atoms prefixed fz-, which change only font size, mainly used for inline (text-level) elements. The second lists atoms prefixed fzb-, for font-size/line-height combos, which are useful for maintaining vertical rhythm on block elements.

Notice there are two pairs of classes for each size. The first classname has an SI prefix that corresponds to a power of 1000, while the second classname (highlighted) uses a Greek/Latin prefix that indicates the value of font-size in its definition. The first set is good for comparing two different atoms, and the second set is good for when you need to know the absolute size.
